Description

This class implements methods for manipulating buffer containing source code in some programming language.

INSTANCE ATTRIBUTES

STR file_name=None
Name of the source file loaded into buffer
STR language=None
Name of language of the source file
INT cur_pos=0
Cursor position (in number of chars) in the buffer
INT visible_start=None
Start position (in number of chars) of the visible portion of the source buffer
INT visible_end=None
End position (in number of chars) of the visible portion of the source buffer
STR content=None
Content of the source buffer
