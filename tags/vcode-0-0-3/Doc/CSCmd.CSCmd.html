Description

A CSC is a phrase which, when uttered into an application, may fire a particular action.

A CSC may fire different actions depending on the context of the application where it was typed.

INSTANCE ATTRIBUTES

STR spoken_forms=[]
list of alternatives ways that this command can be spoken out. They can be regular expressions.
meanings=[[ Context , FCT]]
list of possible contextual meanings for this command. Each element is a pair with 1st element being a context and the 2nd element being an action function to be fired if that context applies.
