Description

abstract base class describing additional interfaces for a (hidden) speech buffers, used to keep track of dictated text and context.

Note: Generally, a concrete class will inherit from TextBuffer (or a subclass) and often, either VisibleBuffer or SpeechBuffer. SpeechBuffer does not inherit from TextBuffer so as to simplify such mix-and-match multiple inheritance.

TextBuffer.SpeechBuffer.set_lock(self, state)

None

locks/unlocks changes to the contents of a hidden speech buffer, to ensure consistency between multiple get operations. When the buffer is locked, all speech-initiated changes to the buffer will be deferred until it is unlocked. No speech should be lost.
